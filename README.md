# nodelib_rollup_ts

基于 typescript 开发的 nodejs 库模板

使用 rollup 构建

使用 eslint + prettier 格式化

使用 husky + lint-stage 标准化流程

使用 ts-jest 完成测试

使用 githubCI 自动化集成

## 创建项目

使用`cofa`:

`cofa create ${your project name}`

`cd ${your project name}`

`npm install`

不使用：

`git clone ${this repo url}`

`cd ${this repo name}`

`npm install`

## 开发流程

开发代码

`npm run dev`

提交代码并合入
